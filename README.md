S’han afegit dos formularis a la web:

El projecte s’ha realitzat de manera col·laborativa entre els següents membres:

- Lucas: Maquetació general de la web i estructura HTML de les pàgines principals.
- Álvaro: Implementació i estilització dels formularis (login i contacte), així com la validació i accessibilitat.
- Iker: Disseny visual, estils CSS generals i adaptació responsive del lloc web.

Tots els membres han participat en la revisió final del projecte.

S'ha creat la seguent informació:

- Un formulari d’accés d’usuari, situat en una pàgina independent, amb els camps d’usuari i contrasenya.
- Un formulari de contacte, visible per a l’usuari, amb camps de nom, correu electrònic i missatge.

Els camps utilitzen etiquetes <label> per millorar l’accessibilitat i validació HTML5 mitjançant l’atribut required.  
Els formularis s’han estilitzat amb CSS utilitzant Flexbox per garantir una estructura clara i adaptada a dispositius mòbils, mantenint l’estètica general de la web.
