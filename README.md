**CSS**
**Descripción del diseño visual triat**
El dobladillo tiene un diseño moderno y sencillo, de color blanco principal y taronja de segundos, utilizando las fuentes Arial, Helvética y sans-serif.

**Evidencia del diseño adaptativo:**
Mitjançant media queries: permite aplicar estilos en diferentes secciones de la pantalla de pantalla, diseñar pantallas flexibles en %, imágenes que se ajustan en elementos que se reordenan en grids y flexbox: facilita que el contingut se adapte y fluya de manera natural en diferentes dispositivos.

**Formularis**
S'han afegit dos formularis a la web:

- Un formulari d'accés d'usuari, situat en una pàgina independent, amb els camps d'usuari i contrasenya.
- Un formulari de contacte, visible per a l'usuari, amb camps de nom, correu electrònic i missatge.

La validació s’ha controlat mitjançant HTML5, utilitzant l’atribut required i tipus d’entrada específics com email, permetent que el navegador verifiqui les dades abans de l’enviament.  
Els formularis s'han estilitzat amb CSS utilitzant Flexbox per garantir una estructura clara i adaptada a dispositius mòbils, mantenint l'estètica general de la web.
